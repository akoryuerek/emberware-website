# Emberware Website

Landing Page für **Emberware** - TV Management System für Sport-Venues.

## Tech Stack

- **HTML5** + **Tailwind CSS** (CDN)
- **Azure Static Web Apps** (Hosting)
- **Custom Domain:** www.emberware.de

## Struktur

```
.
├── index.html           # Hauptseite (Hero, Features, Use Cases)
├── impressum.html       # Impressum (§5 TMG)
├── datenschutz.html     # Datenschutzerklärung (DSGVO)
├── agb.html            # Allgemeine Geschäftsbedingungen
├── emberware-logo.png  # Logo
└── screenshot-board.svg # Dashboard Mockup
```

## Deployment

Automatisch via GitHub Actions → Azure Static Web Apps bei Push zu `main`.

## Lokale Vorschau

```bash
open index.html
```
